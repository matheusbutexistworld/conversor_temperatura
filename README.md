# conversor_temperatura
Conversor de Temperatura simples de Celsius para Fahrenheit.

O script utiliza a informação do número que o usuário e já faz a conversão automática do valor inserido que é considerado no script como "Celsius", e logo após ele faz uma conversão utilizando uma lógica (fórmula) matemática para a conversão, e o resultado desta conversão é exibida no terminal, tanto o valor de entrada (Celsius) e de saída (Fahrenheit).

# Requisitos
# Ambiente 
Ter o Python instalado
# Ferramentas
Executar via Terminal (CMD ou PowerShell)
# Utilização
Usuário digitar um número para a inicialização da conversão, quando o terminal solicitar.
